v29 — Максимум удобства (без "фантазийных" поправок по умолчанию)

Главное:
- Интерфейс разделён на вкладки: Тактика / Погода-ACE / Орудия / Настройки.
- Отдельное окно "Решение огня" (Always on Top), удобно держать поверх игры.
- 3 орудия (вкладки) + расчёт для всех.
- Блокировка ввода координат орудий (чтобы случайно не перезаписать).
- Миникарта: загрузка, калибровка по 2 точкам, постановка цели/орудий кликом.
- Линейка-азимут на карте (улитка): 2 клика -> дистанция (м) и азимут (mil).
- Корректировка R/L/A/D:
  - от орудия → цель
  - от дрона → цель
  - от наблюдателя → цель
  - два наблюдателя: пересечение азимутов (использует позиции наблюдателя и дрона как 2 точки + 2 азимута)

ACE/атмосфера/спин:
- Вкладка "Погода / ACE" содержит поля, но они не применяются автоматически.
- По твоему требованию "не добавлять то, чего нет в игре" — они выключены по умолчанию.
  Если у тебя есть конкретный мод/режим с реальным влиянием — можно включить и привязать к изменению drag.

Удалённая передача/телефон:
- Реалистичный путь: сделать маленький HTTP-сервер в калькуляторе + web-страницу на телефоне (в одной Wi‑Fi сети).
  Это лучше, чем отдельное приложение, и делается без магазина/подписи.
